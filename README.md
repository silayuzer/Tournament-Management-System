# 🏆 Tournament Management System

A full-stack web application built with Django that allows users to create, manage, and participate in tournaments with team registration and automated match generation.

---

## 🚀 Features

- User authentication (custom email-based login)
- Team registration system
- Tournament creation and management
- Application deadline & max team validation
- Admin panel controls
- Match generation system
- Password reset functionality
- User feedback with Django messages

---

## 🛠 Tech Stack

- Python
- Django
- SQLite
- HTML / CSS

---

## ⚙️ Setup

1. Clone the repo  
2. Create virtual environment  
3. Install dependencies:
pip install -r requirements.txt
4. Run migrations:
python manage.py migrate
5. Start server:
python manage.py runserver

---

## 🧠 Key Features

- Prevents duplicate team registrations
- Enforces max team limits
- Handles tournament deadlines
- Role-based access (admin/user)
